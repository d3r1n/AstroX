## Controls

#### ***Movement:***

**W** | **Up Arrow** - Forward \
**A** | **Left Arrow** - Left \
**S** | **Down Arrow** - Backward \
**D** | **Right Arrow** - Right

#### ***Shooting:***

**Space** - Shoot
